¿Que aprendí en este curso?

En este curso aprendí las bases del HTML,CSS Y JavaScript, a utilizar GitHub,sincronizarlo a nuestra computadora y crear repositorios La razon por la que quise ser parte de este curso fue para ponerme a prueba a mi misma y explorar este mundo de la programacion.
Fue todo un reto hubieron noches que no dormia pensando que estaba mal en mi codigo o como mejorarlo,pero cada vez que lograba que algún codigo resultara , fue la mayor satisfacción esto principalmente me enseño a tener mucha paciencia, porque algunas veces somos impacientes de querer todo rápido y programar requiere tiempo y detalle ,un paréntesis mal posicionado puede cambiar todo,mi forma de ver las paginas web cambiaron mucho,
ahora entiendo que al querer agregar color y una imagen a cada pagina requiere un buen uso de CSS o que al querer darle función a cada botón requiere saber utilizar JavaScript pero que la base de todo esto es el HTML tambien aprendimos a utilizar github y crear repositorios, lo más dificil para mí fue JavaScript pero me dedicare a practicarlo más y poder dominarlo a futuro.
